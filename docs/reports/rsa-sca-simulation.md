# Overview
Implement a Side-Channel Analysis (SCA) simulation environment for 16-bit RSA and a 3-LFSR on macOS. The goal is to perform Differential/Correlation Power Analysis (DPA/CPA) using Hamming Distance (HD) models extracted from VCD files generated by GHDL, targeting the recovery of the RSA secret key ($d$) and LFSR states.

# Project Type
BACKEND (Hardware Simulation & Cryptanalysis)

# Success Criteria
- [ ] 16-bit RSA algorithm implemented in VHDL with existing `mon_pro` and `mod_exp` modules.
- [ ] 3-LFSR implemented in VHDL.
- [ ] GHDL simulation successfully generates VCD files containing internal state transitions.
- [ ] Python SCA script successfully parses VCD files, computes Hamming Distance (HD) power models.
- [ ] Python CPA/DPA algorithm successfully recovers the secret key $d$ (RSA) and $k$ (ECC/LFSR) from the simulated power traces.

# Tech Stack
- **Hardware Description:** VHDL
- **Simulation:** GHDL (macOS compatible)
- **Waveform Analysis:** WaveTrace / GTKWave
- **Cryptanalysis & Automation:** Python 3 (NumPy, SciPy, pandas, pyvcd for VCD parsing)
- **OS:** macOS
